🔊 Choose Voice

Tap a voice to select it and hear a sample.

Don't see a male Mexican voice? Install one (free, takes ~1 min)

iPhone / iPad:

  1. Settings → Accessibility → Spoken Content → Voices
  2. Tap Spanish → Spanish (Mexico)
  3. Tap a male voice (e.g. Jorge) and download it
  4. Come back here and pull down to refresh the page — the voice will appear above

Mac:

  1. System Settings → Accessibility → Spoken Content
  2. Click System Voice → Manage Voices…
  3. Find Spanish (Mexico), download a male voice (e.g. Jorge)
  4. Reload this page — it'll appear in the list above
